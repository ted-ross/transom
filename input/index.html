Apache Qpid

Messaging built on AMQP

Qpid makes messaging tools that speak AMQP and support many languages and platforms.

AMQP is an open internet protocol for reliably sending and receiving messages. It makes it possible for us and others to build a diverse, coherent messaging ecosystem.
