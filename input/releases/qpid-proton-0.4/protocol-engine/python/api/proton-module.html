Module proton

source code

The proton module defines a suite of APIs that implement the AMQP 1.0 protocol.

The proton APIs consist of the following classes:

  • Messenger -- A messaging endpoint.
  • Message -- A class for creating and/or accessing AMQP message content.
  • Data -- A class for creating and/or accessing arbitrary AMQP encoded data.
